# 🔐 Password Strength Checker (Python + Tkinter)

A simple desktop application that checks how strong a password is and gives suggestions to improve it. Built using Python and Tkinter GUI.

---

## 🚀 Features

- Checks password strength (Weak / Medium / Strong)
- Validates:
  - Length of password
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Detects common weak passwords
- Provides improvement suggestions
- Simple GUI using Tkinter

---

## 🛠️ Technologies Used

- Python
- Tkinter (GUI library)
- Regular Expressions (re module)
